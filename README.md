# Corporate Sales Performance & Profitability Analysis (MySQL)

## Project Overview
This repository contains a suite of professional MySQL queries designed to extract business intelligence from transaction databases. The scripts address retail operations like month-over-month sales trends, regional customer rankings, and product portfolio margin auditing.

## MySQL Techniques Showcased
- **Common Table Expressions (CTEs)**: Used to break down complex calculations into readable, modular blocks.
- **Window Functions (`LAG`, `DENSE_RANK`)**: Applied over partitions to compute comparative running metrics efficiently.
- **Date Manipulation (`DATE_FORMAT`)**: Structured raw dates into standardized monthly intervals for corporate reporting.

## Repository Structure
- `mysql_sales_queries.sql`: Standalone production-ready MySQL script file with clean formatting and analytical pipelines.
